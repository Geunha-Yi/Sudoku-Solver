# 스도쿠 솔버 (Vue 3)

**기초 ~ 최후의 수단**까지 전체 스도쿠 풀이 기법을 지원하는 단계별 솔버입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

## 구현된 기법

| 구분 | 기법 |
|------|------|
| **기초** | 가득찬 유닛, 드러난 하나, 숨겨진 하나 |
| **초급** | 교차로(포인팅/클레이밍), 드러난/숨겨진 둘·셋·넷, X-윙, 황새치, 해파리 |
| **중급** | XY-윙, XYZ-윙, WXYZ-윙, 심플 컬러링, X-사슬, XY-사슬, 핀드 X-윙/스워드피쉬, BUG, 유일성 직사각형 |
| **고급** | ALS-XZ, Sue de Coq |
| **최후** | 강제 사슬, 백트래킹 |

## 사용법

1. **직접 입력**: 9×9 그리드에 1~9 숫자 입력
2. **예제**: 샘플 퍼즐 불러오기
3. **단계별 풀기**: 각 기법이 적용되는 과정을 이전/다음으로 확인
4. **초기화**: 보드 비우기

## 기술 스택

- Vue 3 (Composition API)
- Vite 5
